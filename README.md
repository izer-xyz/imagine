# imagine

Custom openwrt image to run dockerd on RPi 4. 

## Goals

 * Minimal reproductible host OS
 * Easy upgrade

## How

 * OpenWRT base
   * Well maintained and tested
   * Automated build (ImageBuilder + Action)
   * Rich list of packages
   * Secure by default
